## Cách chạy dự án

1. Clone dự án từ GitHub:
   ```bash
   git clone <URL_GITHUB_REPO>

2. Di chuyển vào thư mục dự án:
    - folder frontend: cd frontend 
    - folder backend: cd backend
    - folder backend sẽ cần tạo thêm file .env
3. Cài đặt các dependencies: npm install
4. Chạy dự án: npm run dev
5. Mở trình duyệt và truy cập vào địa chỉ được cung cấp
